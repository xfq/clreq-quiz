# 中文排版问题小测验

一个约 2 分钟完成的中文排版小测验，用来帮助读者、设计师、前端工程师和编辑出版从业者发现中文网页与电子书中常见的排版问题。

题目内容基于 [W3C《中文排版需求》](https://www.w3.org/TR/clreq/) 整理，覆盖标点位置、换行禁则、省略号、破折号、着重号、中西混排、段首缩排等主题。

## 体验地址

https://xfq.github.io/clreq-quiz/

## 贡献题目

欢迎贡献题目、勘误或补充不同地区的排版差异。建议每道题包含：

1. 题目主题，例如“行首行尾禁则”、“省略号”、“中西文混排”。
2. 一个简短场景，最好能模拟真实网页、电子书或Web应用中的问题。
3. 3 个选项，其中只有 1 个正确答案。
4. 解释文字，说明为什么该选项正确。
5. 对应的 clreq 章节链接或其他可靠来源。

可以在项目 Issues 中提交： https://github.com/xfq/clreq-quiz/issues

## 检查

运行静态检查：

```bash
node tests/static-checks.js
```
